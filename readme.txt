#########################Инструкция по использованию#########################
1. Скачать с Jira все отчеты в формате xls в папку с скриптом.
2. Запустить скрипт rename.py. Он переименует все находящиеся в папки файлы с типом xls  в файлы с типом html.
3. Запустить скрипт jira-report.py. 
4. Потождать пока отработает скрипт. По завершении окно программы закроется самостоятельно.
5. После полной отработки в папки с скрептом появится файл jira.xls. Это и есть требуемый отчет.

#########################Системные требования#########################
1. python 2.7.3
2. Библиотеки для python:
- selenium
- xlwt
3. Firefox
